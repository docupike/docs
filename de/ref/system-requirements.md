# Systemanforderungen

Um eine optimale Benutzererfahrung zu gewährleisten, stellen Sie bitte sicher, dass alle Ihre Clients die Softwareanforderungen von docupike erfüllen.

## Unterstützte Webbrowser

docupike ist eine typische moderne Webanwendung. Daher benötigen Sie auf allen Clients einen Webbrowser.

Allgemeine Anforderungen:
- JavaScript und Cookies müssen aktiviert sein.
- Alle modernen Webbrowser sollten funktionieren.
- Desktop-Varianten von Google Chrome und Mozilla Firefox werden aktiv unterstützt (aktuelle Version und letzte langfristig unterstützte Version).
- Je mehr Informationen Sie gleichzeitig sehen möchten, desto höher sollte die Bildschirmauflösung sein. Die Mindestauflösung sollte FullHD (`1920x1080`) betragen.
- Die Betriebsplattform spielt keine Rolle.
