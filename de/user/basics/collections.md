# Sammlungen

Sammlungen sind Zusammenstellungen von Objektklassen. Sie werden vom Systemadministrator definiert, aber docupike wird mit einem vordefinierten Satz von Sammlungen ausgeliefert. Ihr Vorteil besteht darin, dass Klassen in Kategorien eingeordnet werden, die für den Benutzer leichter zu finden sind.

Klassen werden direkt unter dem Menü "Finder" in der Symbolleiste angezeigt, und jeder Benutzer kann Klassen, die nicht häufig verwendet werden, individuell ausblenden. Sie sind für den Benutzer weiterhin über die Schaltfläche "Mehr" unten im Klassendialog verfügbar.

Im Dropdown-Menü "Finder" finden Sie nicht nur Standard-Sammlungstypen, sondern auch zwei spezielle Arten von Sammlungen: "Favoriten" und "Alle Klassen". "Favoriten" enthält die Klassen, die der Benutzer mit einem "Favoritenstern" markiert hat. Die Sammlung "Alle Klassen" zeigt immer alle aktivierten Klassen an.
