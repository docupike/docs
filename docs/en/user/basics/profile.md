# Your profile

The **Profile** page lets every user manage their own account, name, e-mail, password, default tenant, and language.
It is the first sub-page in the **Settings** sidebar (under the *Settings* group).

Open it from the user menu (avatar at the top-right) → **Settings** → **Profile**.

## Account details

The page is a single form with the following fields.

| Field | Notes |
|---|---|
| **First name** | Your given name. |
| **Last name** | Your family name. |
| **E-mail address** | Used as your login and for notifications. Changing the e-mail typically requires a verification step. |
| **Default language** | The UI language that loads on login. The same picker is also available globally via the **EN** dropdown in the top bar. |
| **Default tenant** | The tenant that is automatically selected on login. Only shown if you have access to more than one tenant, see [Switch between tenants](tenant-switcher.md). |

Click **Save** to persist changes.

## Change your password

The **Security** card on the same page lets you change your password.

1. Click **Change password** on the *Security* card.
2. Enter your current password.
3. Enter the new password twice.

Passwords must be at least 8 characters (max 64), contain at least one number, and use a mix of upper- and lower-case letters.

## Sessions and logout

Use **Logout** in the user menu to end the current session immediately.
