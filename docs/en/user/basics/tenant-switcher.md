# Switch between tenants

The **tenant switcher** is the dropdown at the top-left of every page, directly to the right of the *i-doit up* logo.
It shows the name of the tenant you are currently working in (for example **CareBears Inc.**) with the label *Tenant* underneath.

## When the switcher appears

The switcher is shown to users who have access to **more than one** tenant.
Single-tenant users still see the dropdown, but it only contains the one tenant they belong to.

## Switch tenant

1. Click the tenant name at the top-left.
2. The dropdown opens with every tenant assigned to your user, plus the count of objects currently inside each tenant.
3. Click the tenant you want to work in.

The application reloads and shows only data belonging to the selected tenant.
Permissions, classes, and saved filters are scoped per tenant, switching may show a different sidebar, different categories, and different objects.

## Default tenant

Each user has a **default tenant**: the one that is automatically selected on every login.
Open **Settings > Profile** to change your default tenant if you have access to several.

## When you don't see anything

If your account has no tenant assigned, the switcher shows the message *You have no access to any tenant.*
Contact an administrator to be assigned to at least one tenant before you can use the application.
