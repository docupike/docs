# Favorites

Favorites give you quick access to the things you work with most often.
i-doit up supports two kinds of favorites:

- **Attribute favorites**: mark individual attributes inside an object so they appear in a **★ Favorites** entry at the top of the [object details](object-details.md) sidebar.
- **Class favorites**: mark whole classes so their objects appear together in a personal **Favorites** collection in the Finder.

Favorites are personal, your selections do not affect other users.

## Rights

No administrative right is required.
Any user can favor classes and attributes they have access to.

## Favor an attribute

1. Open the object containing the attribute and select the relevant category in the sidebar.
2. Each attribute row has a **★ Add to favorites** star next to its **Edit** pencil.
3. Click the star to add the attribute to your favorites.

The attribute then appears in the **★ Favorites** entry at the top of the object's category sidebar.
Click **Favorites** in the sidebar to see all favored attributes of this object on one page.

To remove an attribute from your favorites, click the filled star again.

## Favor a class

The class favorite control is shown next to the class name in the Finder header **after** you select a specific class, it is not exposed on the class list itself.

To favor a class:

1. In the Finder, click the class in the left sidebar (for example **Server**).
   The URL updates to `/finder?class=<class-id>` and the heading next to the *New object* button changes from *All objects* to the class name (for example **Server**).
2. Click the outlined **★** star icon next to the heading.

The star fills in once the class is favored.
Click the filled star again to remove the class from your favorites.
