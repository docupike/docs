# Suche speichern

Im Dialogfeld „**Suche speichern**“ werden ein Suchbegriff, die aktiven Filter und die aktive [Spaltenvoreinstellung](preset-configuration.md) unter einem Namen gespeichert, sodass Sie die gesamte Kombination später im Finder erneut anwenden können.
Gespeicherte Suchen sind persönlich und für andere Benutzer nicht sichtbar.

## Rechte

Es sind keine Administratorrechte erforderlich.
Jeder Benutzer kann eigene gespeicherte Suchanfragen erstellen, anwenden und löschen.

## Aktuelle Suche speichern

1. Öffnen Sie den Finder.
2. Geben Sie einen Begriff in das Suchfeld (`What are you looking for?`) ein und drücken Sie die **Eingabetaste**, oder wenden Sie Filter aus dem Dialogfeld **Filter** an, oder beides.
3. Sobald die Suche aktiv ist, erscheint neben dem Suchfeld eine Schaltfläche **Speichern**.
4. Klicken Sie auf **Speichern**.

Es öffnet sich das Dialogfeld **Suche speichern** mit den folgenden Feldern:

| Feld | Erforderlich | Anmerkungen |
|---|---|---|
| **Name** | ja | Freitext. Der Platzhalter im Dialogfeld lautet `Type a name`. |
| **Voreinstellung für** | ja | Kombinationsfeld, das die Spaltenvoreinstellung anzeigt, die bei dieser Suche gespeichert wird. Standardmäßig ist die aktuell aktive Voreinstellung ausgewählt (zum Beispiel `default`). |
| **Begriff** |, | Schreibgeschützt. Zeigt den aktuellen Suchbegriff an. |
| **Filter** |, | Schreibgeschützt. Zeigt eine Übersicht der aktiven Filter an oder`-`  , falls keine vorhanden sind. |

Klicken Sie auf **Speichern & anwenden**, um die Suche zu speichern und sofort anzuwenden.
Klicken Sie auf **Abbrechen** oder auf das`×`Symbol oben rechts, um den Vorgang ohne Speichern zu beenden.

## Eine gespeicherte Suche anwenden
