# Ihr Profil

Auf der Seite **Profil** kann jeder Benutzer sein eigenes Konto, seinen Namen, seine E-Mail-Adresse, sein Passwort, den Standard-Mandanten und die Sprache verwalten.
Es handelt sich um die erste Unterseite in der Seitenleiste **Einstellungen** (unter der Gruppe *Einstellungen*).

Öffnen Sie die Seite über das Benutzermenü (Avatar oben rechts) → **Einstellungen** → **Profil**.

## Kontodaten

Die Seite besteht aus einem einzigen Formular mit den folgenden Feldern.

| Feld | Anmerkungen |
|---|---|
| **Vorname** | Ihr Vorname. |
| **Nachname** | Ihr Nachname. |
| **E-Mail-Adresse** | Wird als Login und für Benachrichtigungen verwendet. Eine Änderung der E-Mail-Adresse erfordert in der Regel einen Bestätigungsschritt. |
| **Standardsprache** | Die Sprache der Benutzeroberfläche, die beim Anmelden geladen wird. Die gleiche Auswahl ist auch global über das Dropdown-Menü **EN** in der oberen Leiste verfügbar. |
| **Standard-Mandant** | Der Mandant, der beim Anmelden automatisch ausgewählt wird. Wird nur angezeigt, wenn Sie Zugriff auf mehr als einen Mandanten haben, siehe [Zwischen Mandanten wechseln](tenant-switcher.md). |

Klicken Sie auf **Speichern**, um die Änderungen zu übernehmen.

## Passwort ändern

Über die Registerkarte **Sicherheit** auf derselben Seite können Sie Ihr Passwort ändern.

1. Klicken Sie auf der Registerkarte *Sicherheit* auf **Passwort ändern**.
2. Geben Sie Ihr aktuelles Passwort ein.
3. Geben Sie das neue Passwort zweimal ein.

Passwörter müssen mindestens 8 Zeichen lang sein (maximal 64), mindestens eine Ziffer enthalten und aus einer Kombination aus Groß- und Kleinbuchstaben bestehen.

## Sitzungen und Abmeldung

Verwenden Sie im Benutzermenü die Option **Abmelden**, um die aktuelle Sitzung sofort zu beenden.
