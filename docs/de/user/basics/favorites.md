# Favoriten

Über die Favoriten gelangen Sie schnell zu den Dingen, mit denen Sie am häufigsten arbeiten.
i-doit up unterstützt zwei Arten von Favoriten:

- **Attribut-Favoriten**: einzelne Attribute innerhalb eines Objekts markieren, damit sie im Eintrag **★ Favoriten** oben in der Sidebar der [Objektdetailseite](object-details.md) erscheinen.
- **Klassen-Favoriten**: ganze Klassen markieren, damit ihre Objekte gemeinsam in einer persönlichen **Favoriten**-Sammlung im Finder erscheinen.

Favoriten sind persönlich; Ihre Auswahl beeinflusst andere Benutzer nicht.

## Rechte

Es sind keine Administratorrechte erforderlich.
Jeder Benutzer kann Klassen und Attribute, auf die er Zugriff hat, als Favoriten markieren.

## Ein Attribut bevorzugen

1. Öffnen Sie das Objekt, das das Attribut enthält, und wählen Sie in der Seitenleiste die entsprechende Kategorie aus.
2. Neben dem Bleistift-Symbol für die Bearbeitung befindet sich in jeder Attributzeile ein Stern mit der Aufschrift **★ Zu Favoriten hinzufügen**.
3. Klicken Sie auf den Stern, um das Attribut zu Ihren Favoriten hinzuzufügen.

Das Attribut wird dann im Eintrag **★ Favoriten** oben in der Seitenleiste der Objektkategorie angezeigt.
Klicken Sie in der Seitenleiste auf **Favoriten**, um alle als Favoriten markierten Attribute dieses Objekts auf einer Seite anzuzeigen.

Um ein Attribut aus Ihren Favoriten zu entfernen, klicken Sie erneut auf den ausgefüllten Stern.

## Eine Klasse bevorzugen

Das Steuerelement für den Klassenfavoriten wird **nachdem** Sie eine bestimmte Klasse ausgewählt haben neben dem Klassennamen in der Finder-Kopfzeile angezeigt; es ist nicht in der Klassenliste selbst zu sehen.

Um eine Klasse zu bevorzugen:

1. Klicken Sie im Finder in der linken Seitenleiste auf die Klasse (z. B. **Server**).
   Die URL wechselt zu `/finder?class=<class-id>`, und die Überschrift neben dem Button *New object* ändert sich von *All objects* auf den Klassennamen (zum Beispiel **Server**).
2. Klicken Sie auf das umrandete Sternsymbol **★** neben der Überschrift.

Der Stern wird ausgefüllt, sobald die Klasse zu den Favoriten hinzugefügt wurde.
Klicken Sie erneut auf den ausgefüllten Stern, um die Klasse aus Ihren Favoriten zu entfernen.
