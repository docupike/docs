# Add-ons

Add-ons erweitern i-doit um zusätzliche Funktionen.
Sie können dem Basisprodukt neue Menüeinträge, Widgets, CMDB-Kategorien, Berechtigungen und API-Endpunkte hinzufügen.

Die verfügbaren Add-ons sind im Benutzermenü (oben rechts) unter **Add-ons** aufgeführt.
Sie können dieselbe Liste auch über **Einstellungen > Verwaltung > Add-ons** aufrufen.

## Rechte

Um Add-ons zu verwalten, benötigen Sie die Berechtigung **Rechte > Verwaltung > Add-ons verwalten**.

Jedes Add-on kann zudem eigene Berechtigungen festlegen.
Diese werden nach der Installation des Add-ons unter **Berechtigungen > Add-ons** angezeigt.

Öffnen Sie im Benutzermenü die **Einstellungen** und navigieren Sie zu **Benutzerverwaltung > Rechte**, um die Rechte zu konfigurieren.

## Ein Add-on installieren

!!! Info
    In der SaaS-Version von i-doit up wird die Installation von Add-ons von i-doit GmbH übernommen.
    Wenden Sie sich an den Support, um ein Add-on anzufordern.

Nach der Installation wird das Add-on automatisch registriert, und seine Menüeinträge, Widgets und CMDB-Strukturen stehen sofort zur Verfügung.

## Installierte Add-ons verwalten

Öffnen Sie im Benutzermenü (oben rechts) den Punkt **Add-ons**, um alle installierten Add-ons und ihren aktuellen Status anzuzeigen.

### Ein Add-on aktivieren oder deaktivieren

Wählen Sie ein Add-on aus und aktivieren oder deaktivieren Sie es über den Schalter.
Durch das Deaktivieren eines Add-ons werden dessen Menüeinträge ausgeblendet und seine Funktionen deaktiviert, ohne dass gespeicherte Daten gelöscht werden.

## Rechte an Add-ons

Add-ons beschränken den Zugriff auf ihre Funktionen mithilfe des Standardberechtigungssystems.
Nach der Installation werden die add-onspezifischen Berechtigungen im Abschnitt **Berechtigungen > Add-ons** angezeigt.

Weisen Sie Benutzern, Gruppen oder Rollen Add-on-Rechte auf dieselbe Weise zu wie Standardrechte.
Weitere Informationen finden Sie unter [Rechte und Berechtigungen](rights-and-permissions.md).

## Verfügbare Add-ons

### Datenschutz (DSGVO)

Das Add-on **Datenschutz** hilft Ihnen dabei, Verarbeitungsvorgänge, externe Auftragsverarbeiter und Datenschutzmaßnahmen gemäß den Anforderungen der DSGVO zu dokumentieren.

Nach der Installation erscheint im Menü „Bestand“ der Eintrag **Datenschutz**.
Eine 30-tägige Testversion ist direkt über die Add-ons-Übersicht verfügbar.

### Dokumenten-Ersteller

Das Add-on **Documents Creator** erstellt anhand konfigurierbarer Vorlagen Dokumente aus Ihren CMDB-Daten.
